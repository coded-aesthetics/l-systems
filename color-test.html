üìù Basic Syntax Examples

Single Colored Symbol:

Rule with Colors:

  • Parser: Uses regex-based tokenization to extract symbols and parameters
  • Color Support: RGBA with full alpha channel support
  • Fallback: Invalid colors fall back to default system colors
  • Performance: Colors are parsed once during rule interpretation
  • WebGL: Colors are passed as vertex attributes for efficient GPU rendering
  • Compatibility: Fully backward compatible with existing L-systems
